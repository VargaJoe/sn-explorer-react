# Story 05: File and Folder Operations (CRUD)

**As a user, I want to create, rename, move, copy, and delete files and folders, so I can organize my content as needed.**

## Acceptance Criteria
- Context menu and toolbar actions for create, rename, move, copy, delete.
- Supports drag & drop for move/copy.
- Confirmation dialogs for destructive actions.
- Operations work for all container/content types where applicable.
- UI updates in real time after operations.
