# Story 02: Navigation - Breadcrumbs

**As a user, I want to see and use a breadcrumb navigation bar, so I can understand my current location and quickly jump to any parent folder or container.**

## Acceptance Criteria
- Breadcrumbs reflect the current path in the repository.
- Each segment is clickable and navigates to that level.
- Supports all container types, not just folders.
- Updates in sync with tree and main view navigation.
