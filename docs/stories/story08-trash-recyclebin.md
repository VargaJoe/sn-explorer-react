# Story 08: Trash / Recycle Bin

**As a user, I want to recover deleted items from a trash or recycle bin, so I can restore content that was removed by mistake.**

## Acceptance Criteria
- Deleted items are moved to a trash container, not permanently deleted.
- Trash is accessible from the navigation tree.
- Items can be restored or permanently deleted from trash.
- Supports bulk restore and delete.
