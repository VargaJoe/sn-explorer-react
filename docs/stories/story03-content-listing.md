# Story 03: Content Listing

**As a user, I want to see a list of all items (files, folders, and other content types) in the selected container, so I can browse, select, and interact with them.**

## Acceptance Criteria
- Main area displays a table or grid of items in the current container.
- Shows name, type, icon, modified date, and other relevant fields.
- Supports all SenseNet content types, not just files and folders.
- Double-clicking a folder navigates into it.
- Supports sorting and basic filtering.
