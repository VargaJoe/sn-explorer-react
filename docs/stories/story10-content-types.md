# Story 10: Support for All SenseNet Content Types

**As a user, I want to browse, create, and manage all types of content supported by SenseNet (not just files and folders), so I can leverage the full power of the ECM.**

## Acceptance Criteria
- UI supports all major SenseNet content types (e.g., File, Folder, Image, List, Task, User, Group, Workspace, SmartFolder, etc.).
- Creation dialogs/forms adapt to the selected content type.
- Icons and metadata reflect the type of each item.
- Filtering and search work across all types.
