# Story 04: Search

**As a user, I want to search for content by name, type, or metadata, so I can quickly find what I need in large repositories.**

## Acceptance Criteria
- Search bar is available at the top of the main view.
- Supports searching by name, type, and custom fields.
- Results update the content list and highlight matches in the tree.
- Search is fast and works across all content types.
